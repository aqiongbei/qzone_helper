## 原理简介

利用插件向页面注入批处理逻辑，通过模拟前端请求实现自动删除功能，过程中不需要用户参与交互，操作流畅迅速，**不会出现验证码**，同时方法安全可靠亦**不会造成封号风险**。但是频繁操作仍会触发后端限制逻辑,使用时候须留意.
